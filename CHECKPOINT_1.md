## 1. Архитектурный план и концепция

### Цель сервиса

**HealthChecker** — это микросервис для мониторинга доступности сайтов и API.

Сервис по расписанию проверяет URL-адреса из файла `config.yaml`, анализирует HTTP-статус, время ответа и состояние SSL-сертификата.

При обнаружении ошибки, таймаута, неожиданного HTTP-статуса или проблемы с SSL сервис записывает событие в лог и отправляет уведомление в Telegram.

### Целевой интерфейс

Фоновый сервис с консольным запуском через команду:

```powershell
dotnet run
```

### Выбранный стек технологий

- **Язык программирования:** C# .NET 8
- **Тип приложения:** .NET Worker Service
- **HTTP-клиент:** HttpClient
- **Конфигурация:** YamlDotNet, `config.yaml`
- **Уведомления:** Telegram.Bot
- **Логирование:** NLog
- **Статический анализ:** StyleCop Analyzers
- **Контейнеризация:** Docker, Docker Compose

## 2. Проектирование

### Диаграмма вариантов использования

```mermaid
flowchart TD
    User["Пользователь / администратор"] --> Config["Настраивает config.yaml"]
    User --> Run["Запускает HealthChecker"]

    Run --> Worker["HealthCheckWorker"]
    Worker --> Check["Проверка сайтов и API"]

    Check --> HTTP["Проверка HTTP-статуса"]
    Check --> Time["Измерение времени ответа"]
    Check --> SSL["Проверка SSL-сертификата"]

    HTTP --> Result["Формирование результата проверки"]
    Time --> Result
    SSL --> Result

    Result --> Logs["Запись результата в логи"]
    Result --> Alert["Telegram-уведомление при ошибке"]
```

### Диаграмма последовательности взаимодействия модулей

```mermaid
sequenceDiagram
    participant User as Пользователь
    participant Worker as HealthCheckWorker
    participant Config as ConfigLoader
    participant Checker as EndpointChecker
    participant SSL as SslChecker
    participant TG as TelegramNotifier
    participant Log as NLog

    User->>Worker: Запуск dotnet run
    Worker->>Config: Загрузка config.yaml
    Config-->>Worker: Настройки и список URL

    Worker->>Checker: Проверить URL
    Checker->>SSL: Проверить SSL-сертификат
    SSL-->>Checker: Результат SSL-проверки

    Checker-->>Worker: Результат проверки сайта
    Worker->>Log: Записать результат в лог

    alt Обнаружена ошибка
        Worker->>TG: Отправить Telegram-уведомление
        TG-->>User: Сообщение об ошибке
    end
```
