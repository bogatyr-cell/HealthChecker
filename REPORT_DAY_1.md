# ОТЧЕТ ПО ПРАКТИКЕ: ДЕНЬ 1

## 1. ОБЩАЯ ИНФОРМАЦИЯ

**Тема проекта:** HealthChecker: сервис распределенного мониторинга API

**Формат работы:** Индивидуально

**Состав команды и роли:**
- Соло — Backend/DevOps-разработчик. Отвечает за разработку микросервиса, настройку проверки API-эндпоинтов, логирование, конфигурацию, SSL-проверку и Telegram-уведомления.

## 2. ВЫБРАННЫЙ СТЕК ТЕХНОЛОГИЙ

**Язык программирования:** C# .NET 8

**Основной фреймворк/библиотеки:** .NET Worker Service, HttpClient, YamlDotNet, Telegram.Bot

**Система логирования:** NLog

**Линтер / статический анализ:** StyleCop Analyzers
